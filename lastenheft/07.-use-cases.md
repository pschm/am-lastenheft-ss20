| Ausnahmeszenarien  | 2b Kein Algorithmus konfiguriert <br/> 8a Fehler bei Datenauswertung <br/> 8a1 System schreibt Systemstatus ins Log <br/> 8a2 System zeigt Fehlermeldung an  |
