## Umfrage Studenten

- Das System sollte die Vernetzung von Maschinen und Prozessketten und Verarbeitung von Daten verwirklichen. 

- Das System sollte Projekte und deren Ergebnisse in verständlicher Form offen legen.  

- Das System sollte eine Anbindung zu Process Mining und Business Process Automation Lösungen erlauben.

- Das System sollte Variable Software zur Analyse großer Datenmengen, komfortable Tools zum Austausch/zur Transformation von Daten zwischen verschiedenen Plattformen / Programmier-Paradigmen, z.B. relationale DB, R, Java, Python, TensorFlow/Keras. Hardware-Unterstützung bei der Analyse größerer Datenmengen, z.B. GPU-Server etc. bieten.

- Das System sollte Trainingsdaten gelabelled/ungelabelled für supervised/unsupervised learning aus unterschiedlichen Domänen speichern.
